# SonarQube Report

## Mục đích

SonarQube được sử dụng để kiểm tra chất lượng mã nguồn của dự án EventSphere.

---

# Các tiêu chí đánh giá

- Bugs
- Vulnerabilities
- Code Smells
- Security Hotspots
- Reliability
- Maintainability
- Duplicated Code
- Coverage

---
